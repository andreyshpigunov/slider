# Slider

Changes the img image when you swipe over it with the mouse.

## Demo

[andreyshpigunov.ru/slider/](https://andreyshpigunov.ru/slider/)

## Usage

### HTML

```
<div data-slides='["file1","file2",...,"fileN"]'>
  <img src="file1" alt="" />
</div>
```

### JS

```
import Slider from '../components/slider.js';
Slider.init();
```
